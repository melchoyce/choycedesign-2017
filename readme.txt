## Setup

Install the theme by checking it out locally. The built CSS is tracked in git, so no need to build just to use the theme. For development, build the styles with Sass CLI:

	scss assets/stylesheets/style.scss:style.css

Can optionally add `--watch` to automatically run on file change.
